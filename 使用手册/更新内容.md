# 升级日志


> # <i class="fas fa-exclamation-triangle"></i>注
> 如果您从较老版本的小书匠升级，内置数据库会有不兼容问题，建议在升级前进行数据导出备份，或者数据库文件备份，防止升级失败。
> 
> **数据库文件路径**
> 
> ``` 
> Windows: %LOCALAPPDATA%/storywriter/ 
> Mac: ~/Library/Application
> Support/storywriter/ Linux: ~/.config/storywriter 
> ```
